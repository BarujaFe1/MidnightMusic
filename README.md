## 🇧🇷 PT-BR

## 🌙 Visão geral

**Midnight Music Suite** é um aplicativo desktop com interface moderna em modo noturno para organização, extração e conversão autorizada de áudio em formato MP3.

O projeto combina uma GUI elegante feita com **CustomTkinter**, processamento de mídia com **FFmpeg** e uma experiência simples para quem precisa lidar com arquivos de áudio, capas, metadados e conversões locais.

A proposta é oferecer uma ferramenta prática para uso pessoal e autorizado, com foco em:

- conversão para MP3;
- organização local de arquivos;
- extração e aplicação de capas;
- configuração persistente;
- execução em segundo plano sem travar a interface;
- experiência desktop visualmente agradável.

> **Objetivo:** transformar tarefas manuais de áudio em um fluxo desktop simples, bonito e organizado.

---

## 🎯 Problema que resolve

Organizar músicas, áudios, capas e metadados localmente costuma exigir várias ferramentas separadas. Em muitos casos, o usuário precisa alternar entre conversores, editores de tags, gerenciadores de arquivos e scripts improvisados.

Isso gera atrito em tarefas como:

- converter arquivos para MP3;
- manter padrão de nome;
- adicionar metadados;
- salvar capas;
- escolher pasta de destino;
- acompanhar downloads/conversões em lote;
- evitar travamento da interface durante operações longas.

O **Midnight Music Suite** reúne essas rotinas em uma interface única, escura, direta e amigável.

## 🇺🇸 English

## 🌙 Overview

**Midnight Music Suite** is a desktop application with a modern dark interface for authorized audio organization, extraction and MP3 conversion.

The project combines an elegant **CustomTkinter** GUI, media processing with **FFmpeg** and a simple experience for users who need to manage audio files, artwork, metadata and local conversions.

The goal is to provide a practical tool for personal and authorized use, focused on:

- MP3 conversion;
- local file organization;
- artwork extraction and embedding;
- persistent settings;
- background execution without freezing the interface;
- visually polished desktop experience.

> **Goal:** turn manual audio tasks into a simple, beautiful and organized desktop workflow.

---

## 🎯 Problem solved

Organizing music, audio files, artwork and metadata locally often requires multiple separate tools. Users may need to switch between converters, tag editors, file managers and improvised scripts.

This creates friction in tasks such as:

- converting files to MP3;
- keeping filenames consistent;
- adding metadata;
- saving artwork;
- choosing destination folders;
- tracking batch conversions;
- keeping the interface responsive during long operations.

**Midnight Music Suite** brings these routines into a single, dark, direct and user-friendly interface.

## 🚀 Quick Start / Início rápido

### Requirements / Pré-requisitos

- Python 3.10+
- FFmpeg and ffprobe
- Internet connection if using online metadata or authorized URL-based workflows

### Installation / Instalação

```bash
git clone https://github.com/BarujaFe1/Midnight-Music-Suite.git
cd Midnight-Music-Suite
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

```bash
# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
python MidnightMusic.py
```

---

## 🎧 FFmpeg setup / Configuração do FFmpeg

The app depends on FFmpeg for audio conversion and metadata handling.

### Windows

1. Download FFmpeg from the official website or a trusted build provider.
2. Extract the archive.
3. Locate:
   - `ffmpeg.exe`
   - `ffprobe.exe`
4. Place both executables in the same folder as `MidnightMusic.py`, or configure their path in the project if supported.
5. Launch the app and check the dependency status indicator.

### Linux

```bash
sudo apt install ffmpeg
```

### macOS

```bash
brew install ffmpeg
```

## 🎮 Como usar / How to use

### Música / Music

1. Open the music tab.
2. Select or provide authorized media input.
3. Choose the output configuration.
4. Start conversion.
5. The MP3 will be saved in the configured folder.

### Lote / Batch

1. Open the batch tab.
2. Add multiple authorized items.
3. Configure delay and retry attempts.
4. Select which items to process.
5. Start the batch operation.
6. Follow progress in the interface.

### Capas / Artwork

1. Open the artwork tab.
2. Select the media or provide metadata.
3. Save the artwork as JPG or embed it into the MP3 when available.

### Configurações / Settings

1. Choose output folder.
2. Configure delay between operations.
3. Configure retry attempts.
4. Save settings.
5. Restore defaults if needed.

---

## ⚙️ Configuration file / Arquivo de configuração

The app creates `midnight_config.json` automatically.

Example:

```json
{
  "download_path": "D:\\Musicas\\Midnight",
  "delay_between_songs": 5,
  "retry_attempts": 3
}
```

| Key | Description |
|---|---|
| `download_path` | Folder where MP3 files are saved |
| `delay_between_songs` | Delay in seconds between batch operations |
| `retry_attempts` | Number of attempts per item after failure |
